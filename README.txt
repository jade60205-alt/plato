HOW TO PUBLISH (3 quick ways)

1) GitHub Pages (free)
- Create repo: yourname.github.io
- Upload all files
- Visit https://yourname.github.io/index_match_design_nophoto.html
- (Optional) Rename index_match_design_nophoto.html to index.html to use root path

2) Netlify (fastest)
- Go to https://app.netlify.com/drop
- Drag the folder contents to upload
- Site URL will be something like https://<random>.netlify.app
- Set index page: rename index_match_design_nophoto.html to index.html if you want root home

3) Squarespace / Webflow
- Create a blank page
- Add a "Code" component
- Open the HTML file in a text editor, copy all, paste into the code block
- Upload images (logo.png, WechatIMG10785.jpeg) to the asset manager if needed and update paths
